# Getting started

Below mentioned are the steps to install, configure and run this app.

## Installation and configuration

```bash
# clone the project.
https://github.com/Joao-MCF/pokedex.git

# enter the cloned directory.
cd pokedex

# install the necessary packages
npm install

# run the app.
npm run dev
```
